O que é HTML

Voltar

HTML é uma linguagem de marcação de hipertexto utilizada na criação de documentos e páginas da web. Os marcadores, chamados de “tags”, servem para indicar a função de cada elemento na página, como textos, imagens e vídeos, além de suas conexões com outros elementos (links) e como eles serão interpretados pelo navegador.

Os documentos HTML são arquivos de texto simples, com a extensão .html, e podem ser escritos na maioria dos editores de texto comuns, como o Bloco de Notas, do Windows. Em aplicações mais avançadas, recomenda-se a utilização de ambientes de desenvolvimento integrado.

O que são Tags

Voltar

As tags descrevem a estrutura semântica de uma página, ou seja, a organização de suas informações, que então é recebida, interpretada e renderizada pelo navegador.

São instruções que informam ao navegador como formatar o texto.

Marcação de texto

Voltar

Vamos conhecer agora alguns tipos de marcação do texto:

  1. mark

    2. Ele é geralmente usado para destacar palavras-chaves em um resumo, nomes de produtos em um comentário ou outros vãos de texto cuja a apresentação típica seria negrito.

    Foi apresentado um exemplo no começo dessa explicação. Mas segue outra para ficar mais exemplificado.

  2. small

    3. "Small Element" ou fonte menor é utilizado para reduzir o tamanho da fonte num intervalo definido de um texto.

    Você pode encontrar o small em todas as aplicações do "voltar".

  3. i

    4. Pode representar uma parte do texto com uma diferença semântica, na qual a representação tipográfica padrão é no tipo itálico.

  4. u

    5. "Underline Element" ou sublinhado é utilizado para sublinhar um espaço definido de um texto.

    Um exemplo de elemento u é todo esse texto.

  5. strong

    6. O elemento "strong" dá ao texto uma forte importância, e é tipicamente mostrado em negrito.

    O elemento strong foi aplicado em todas as listas.

  6. sub

    7. "Subscript Element" ou subscrito é utilizado escrever um texto com letras menores e com o alinhamento abaixo das que do resto do conteúdo.

    O texto fica assim.Esse é um exemplo de elemento sub. Repare no Texto de exemplo abaixo.

  7. sup

    8. "Superscript Element" ou sobrescrito é utilizado escrever um texto com letras menores e com o alinhamento acima das que do resto do conteúdo.

    Não aparenta ter muita diferença, não é mesmo.E esse é um exemplo de elemento sup. Mas repare no exemplo acima.

  8. blockquote

    9. Indica que o texto incluído é uma longa citação. Normalmente, este é processado visualmente pelo recuo.

    “A vida é 10% do que acontece comigo e 90% de como eu reajo a isso”
    Charles Swindoll.
    “A mente é tudo. Você se torna aquilo que você pensa”
    Buda.

Listas

Voltar

Lista é um importante recurso de HTML, pois permite criarmos tópicos de textos para uma melhor exemplificação de um determinado assunto. São recursos extremamente usados, inclusive quando nem imaginamos que ele esta sendo usado, como no caso de menus.

Ao longo das nossas explicações estamos usando listas ordenadas (ol) e listas não ordenadas (ul).

O li é utilizado nas listas para representar um item.

Outras Tags

Voltar

Vamos conhecer agora outros tipos de tags utilizadas em html.

  1. font

    2. "Font Element" ou letras é utilizado para configurar o tipo, cor e tamanho das letras de um texto de uma página HTML.

  2. del

    3. Representa uma parte do texto que foi excluída de um documento.

  3. abbr

    4. O atributo abbr é usado para fornecer uma breve informação para tecnologias assistivas tais como leitores de tela quando estiverem "lendo" conteúdos tabulares.
